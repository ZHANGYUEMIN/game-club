# 游戏社团官网

这是一个现代化的游戏社团官方网站，采用响应式设计，为游戏爱好者提供信息展示和交流平台。

## 功能特点

### 🎨 视觉体验
- 🎮 **完全响应式设计**，完美适配手机、平板、电脑、大屏设备
- 🌙 **6种炫酷主题**，支持实时切换和本地保存
  - 🟢 赛博绿 - 经典科技风格
  - 🟣 霓虹紫 - 神秘赛博朋克
  - 🔵 电光蓝 - 未来科幻风格  
  - 🟠 烈焰橙 - 激情电竞风格
  - 🩷 神秘粉 - 梦幻电竞风格
  - 💚 黑客绿 - 极客黑客风格
- ✨ **智能粒子背景效果**，根据设备性能和主题自动调整
- 🌈 **全屏霓虹灯特效**，扫描线动画，旋转光晕等炫酷效果
- 📱 横屏模式专项优化

### 💻 核心功能
- 📱 **多页面导航系统**（首页、关于我们、社团活动、成员展示、加入我们、联系我们）
- 📝 **智能表单系统**，包含完整验证和自动邮件通知
- 🏆 **实时排行榜**，支持多游戏切换（LOL、Valorant、CS:GO）
- 👥 **成员展示系统**，包含核心成员和优秀成员介绍
- 📅 社团活动展示和公告中心
- 📞 完整的联系方式和QQ群二维码

### 🚀 用户体验
- 🔔 **智能通知系统**，支持多种类型通知
- ⚡ **性能优化**，包含懒加载、资源预加载、网络适配
- 🎯 **无障碍访问**支持，键盘导航友好
- 📶 智能网络检测，低速网络自动优化
- 🔄 页面可见性检测，后台自动暂停动画节省性能

## 技术栈

### 前端技术
- **HTML5** - 语义化标签，提升SEO和可访问性
- **CSS3** - 现代化样式，包含Flexbox、Grid、动画
- **JavaScript ES6+** - 现代JavaScript特性
- **响应式设计** - Mobile-First设计理念

### 第三方库
- **particles.js** - 动态粒子背景效果
- **EmailJS** - 无服务器邮件发送服务

### 部署平台
- **GitHub Pages** - 静态网站托管

### 性能优化
- **懒加载** - 图片和资源按需加载
- **预加载** - 关键资源提前加载
- **网络自适应** - 根据网络状况调整体验
- **可见性API** - 后台性能优化

## 页面说明

### 📄 核心页面

1. **🏠 首页**
   - 社团简介和愿景展示
   - 三大核心板块（电竞比赛、游戏交流、技术培训）
   - **实时排行榜系统**（多游戏支持）
   - 快速加入按钮

2. **ℹ️ 关于我们**
   - 社团历史和发展历程（2024年成立）
   - 社团文化和理念
   - 组织架构介绍

3. **📅 社团活动**
   - 最新公告和通知
   - 定期活动安排（每周二、三、五）
   - 活动回顾和精彩瞬间

4. **👥 社团成员**
   - **核心成员展示**（社长、副社长、部长等）
   - 优秀成员成就榜
   - 成员技能标签系统

5. **✋ 加入我们**
   - **智能在线申请表单**
   - 实时表单验证（手机号、邮箱、年龄等）
   - 自动邮件通知系统
   - 即时反馈通知

6. **📞 联系我们**
   - 多渠道联系方式
   - QQ群二维码展示
   - 社交媒体链接

### 💡 特色功能

- **🏆 排行榜系统**：支持英雄联盟、Valorant、CS:GO三款游戏
- **🔔 通知中心**：智能弹窗通知，支持成功、信息、警告等多种类型
- **📱 设备适配**：针对手机、平板、电脑、大屏等设备专项优化
- **🎨 动效系统**：页面切换、悬停特效、加载动画等
- **🎭 主题系统**：
  - 点击右侧🎨按钮或按`T`键打开主题面板
  - 快捷键`Ctrl+Shift+数字键(1-6)`快速切换主题
  - 主题选择自动保存到本地存储
  - 粒子效果颜色随主题实时变化

## 本地运行

1. 克隆仓库
```bash

```

2. 进入项目目录
```bash
cd game-club
```

3. 使用浏览器打开 `index.html` 文件
